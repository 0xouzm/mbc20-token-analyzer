### Moltbook API Usage
```bash
# Fetch posts
curl "https://www.moltbook.com/api/v1/posts?limit=100&sort=new"

# Response structure
{
  "posts": [
    {
      "id": "...",
      "title": "...",
      "content": "{\"p\":\"mbc-20\",\"op\":\"mint\",...}",
      "author": {"name": "..."},
      "created_at": "2026-02-12T08:19:57.123456+00:00"
    }
  ],
  "next_offset": 5
}
```

### Token Data Pattern
- Standard mint: `{"p":"mbc-20","op":"mint","tick":"CLAW","amt":"100"}`
- Large batch: `{"p":"mbc-20","op":"mint","tick":"LOBSTER","amt":"1000"}`
- Link wallet: `{"p":"mbc-20","op":"link","wallet":"0x..."}`

### Anomaly Detection Rules
1. **High Volume**: mint_count > 50 in 24h
2. **Large Batches**: avg_amt > 500 per mint
3. **Multi-Author**: unique_authors > 20 (potential botnet)
